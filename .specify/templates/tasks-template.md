## Per-Task contract

Every Task in the table above must satisfy:

1. **Red Test exists first.** Stage 6 of the Crank workflow — test written and failing for the right reason before any implementation code.
2. **Single named test file** at the deterministic path derived from the Story ID.
3. **No new untested dependencies.** If a sub-component is needed, it gets its own Story + Red Test (loop back to `/speckit.tasks` for the sub-Story).
4. **Storybook-green precedes Playwright-green.** Tasks tagged "Feature" do not enter the build until all dependent "Component" and "Page assembly" Tasks are green.

## Definition of done (per Task)

- Red Test now green.
- No regressions in existing tests.
- TypeScript strict passes.
- Tailwind/lint passes.
- Committed with reference to Task # and Story ID.
