## Rule

Use v0 (v0.app) to generate UI from prompts, screenshots, or Figma frames in a chat that is linked to this GitHub repo. v0 auto-creates a `v0/*` working branch and commits on each code-changing message — code enters `main` **only** via a reviewed pull request, never by direct push from v0.

## Common patterns

- Prompt-to-UI generation with shadcn/ui + Tailwind output — [v0 docs](https://v0.app/docs/text-prompting)
- Figma frame import via attached Figma link (reads layout + design tokens) — [v0 docs](https://v0.app/docs/figma)
- Chat linked to GitHub repo; auto-branches from `main`, auto-commits per message, merge via PR — [v0 docs](https://v0.app/docs/github)
- Scaffold a new v0-SDK app with `npx create-v0-sdk-app@latest` — [v0 docs](https://v0.app/docs/api/platform/packages/create-v0-sdk-app)
- Connect MCP servers into v0 generations to ground outputs in project tools/data — [v0 docs](https://v0.app/docs/MCP)
