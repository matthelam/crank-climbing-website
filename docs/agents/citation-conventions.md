# Citation conventions

**Status: *[TBD]*** — scaffolding only.

How to reference phases, keys, and artifacts when reporting state.

## Planned contents

- **Workflow stages** cited by number and Key: "Stage 7 — Recursive Build."
- **Standard Platform tools** cited by Key, bolded as in the README: **Playwright**, **LangGraph**, **Vercel**.
- **Intent artifacts** cited by file path with line number where relevant: `intent/tasks/task-042.md:L17`, rendered as a markdown link.
- **Pull requests and issues** cited as full GitHub links, never bare `#123`.
- **Memory references** cited by title, not file path.

Will be elaborated as concrete artifacts emerge.
